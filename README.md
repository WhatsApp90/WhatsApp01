# بوت واتساب الإسلامي - مجموعة عائلتي العزيزة

بوت واتساب إسلامي ينشر يوميًا أذكار الصباح والمساء لمجموعة "عائلتي العزيزة"، مع التاريخ الهجري وحدث في مثل هذا اليوم، وقصص من بلاد المغرب الإسلامي وجزيرة العرب، ومواقيت الصلاة في ولاية حضرموت، وهمسات إيمانية، وتذكرة بعلماء المسلمين في سجون آل سعود، ودعاء صدقة جارية لصاحب البوت **خَطَّاب الحضرمي** ⚔️🖤.

## التشغيل

1. انسخ `.env.example` إلى `.env` وعدّل القيم:
   - `GROUP_ID`: معرّف مجموعة واتساب (يبدأ بـ `120363...`).
   - `MORNING_TIME` / `EVENING_TIME`: توقيت نشر الأذكار.
2. `npm install`
3. `npm start`
4. امسح رمز QR من الطرفية بربط واتساب.
5. عند نجاح الاتصال، يبدأ النشر التلقائي بتوقيت حضرموت (Asia/Aden).

## الأوامر داخل المجموعة

- `أذكار` - أذكار الصباح
- `مساء` - أذكار المساء
- `مواقيت` - مواقيت الصلاة في حضرموت
- `تاريخ` - التاريخ الهجري
- `حدث` - حدث في مثل هذا اليوم الهجري
- `قصة` - قصة من بلاد المسلمين
- `همسة` - همسة إيمانية
- `دعاء` - دعاء صدقة جارية لصاحب البوت
- `أوامر` - قائمة الأوامر

## النشر التلقائي

- أذكار الصباح: 05:30 (افتراضيًا)
- أذكار المساء: 16:00 (افتراضيًا) — تتضمن مواقيت الصلاة

## ملاحظات

- مواقيت الصلاة تُجلب من Aladhan API لولاية حضرموت (المكلا).
- التاريخ الهجري محسوب تقريبيًا دون مكتبة خارجية.
- ملفات المصادقة تُحفظ في مجلد `auth_info/` (لا ترفعه إلى GitHub).
